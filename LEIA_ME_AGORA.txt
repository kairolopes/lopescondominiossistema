
=== GUIA RÁPIDO: LOPES CONDOMÍNIOS BOT ===

1. O PROBLEMA ATUAL (ERRO DE REDE)
   O sistema tentou criar um link público automaticamente (usando LocalTunnel/Serveo), 
   mas sua rede ou firewall bloqueou todas as tentativas.
   
   Para resolver isso de vez, precisamos usar o NGROK com autenticação.

2. COMO RESOLVER (FAÇA ISSO AGORA)
   1. Na pasta do projeto, dê dois cliques no arquivo:
      configurar_ngrok.bat
      
   2. Uma janela vai abrir e te guiará:
      - Vai abrir o site do Ngrok para você pegar seu token grátis.
      - Você cola o token na janela preta.
      - O sistema vai configurar tudo sozinho.

3. DEPOIS DE CONFIGURAR
   O sistema vai reiniciar e, se tudo der certo, vai mostrar a URL VERDE.
   
   A URL correta será algo como:
   https://xxxx-xx-xx-xx-xx.ngrok-free.app/api/webhook/zapi

4. ONDE COLOCAR ESSA URL?
   - Vá no painel da Z-API > Webhook.
   - Cole a URL inteira (com o /api/webhook/zapi no final).
   - Salve.
